# Techno & EDM Festival 🎧

Landing page de un festival de música electrónica desarrollada como proyecto de práctica frontend. El foco estuvo en diseño responsive, interacciones con JavaScript vanilla y automatización del flujo de trabajo con Gulp.

**Demo en vivo:** https://festival-musica-sebas.netlify.app/

---

## Características

- Header fijo que se adhiere al top al hacer scroll
- Hero con video de fondo en autoplay
- Grilla de line-up con dos escenarios por día
- Galería de imágenes con visor modal y animación de fade
- Sección de boletos/pases con efectos hover
- Navegación con scroll suave y resaltado de enlace activo

---

## Tecnologías



- HTML5 - Estructura
- SCSS + BEM - Estilos 
- JavaScript vanilla - Interacciones DOM 
- Gulp 5 - Compilación SCSS y automatización
- Node.js


## Lo que aprendi

- Arquitectura SCSS con variables y parciales
- CSS Grid y Flexbox para layouts responsive
- Comportamiento de scroll con JavaScript vanilla
- Creación de modales con animaciones CSS (`@keyframes`)
- Gulp como task runner para compilación automática
- Optimización de imágenes y video para web
